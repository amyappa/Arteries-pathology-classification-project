# Arteries-pathology-classification-project
Interpretable analysis and classification of blood flow audio recordings using spectral, temporal, wavelet, nonlinear, and HMM-based features.

# Blood Flow Sound Analysis

This repository contains code and materials for a course project on the analysis of blood flow audio recordings for the detection of vascular pathologies.

The project explores blood flow sounds as time series generated by a complex dynamic system. The main goal is to extract interpretable signal features and evaluate their ability to distinguish normal and pathological recordings.

## Project overview

The analysis includes several groups of features:

* spectral activity in diagnostically relevant frequency bands;
* temporal activity features describing the stability and duration of spectral peaks;
* MFCC-based timbral descriptors;
* wavelet entropy features;
* Lyapunov exponent features for nonlinear dynamics analysis;
* Hidden Markov Model-based state features.

Two classification models were evaluated:

* Logistic Regression as an interpretable linear baseline;
* Gradient Boosting as a nonlinear model capable of capturing feature interactions.

The best performance was achieved by Gradient Boosting with HMM-based features.

## Main results

The best model configuration reached:

* ROC AUC: 0.865
* Recall: 0.714

The results suggest that blood flow audio recordings contain diagnostically relevant information not only in their frequency content, but also in their temporal structure and nonlinear dynamics.

## Methods used

* Fourier transform and spectrogram analysis
* Peak detection in selected frequency bands
* MFCC feature extraction
* Wavelet entropy
* Lyapunov exponent estimation
* Hidden Markov Models
* Logistic Regression
* Gradient Boosting
* Monte Carlo cross-validation
* LIME-based feature importance analysis

## Notes

The dataset is not included in this repository due to data privacy and storage limitations. The code is intended to demonstrate the analysis pipeline and can be adapted to similar `.wav` recordings of blood flow sounds.
